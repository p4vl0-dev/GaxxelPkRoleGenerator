# 🎲 Pokémon/Trainer Generator Module

## This is a fork of [Gaxxel's Pokemon Generator](https://github.com/gaxxel-tech/GaxxelPkRoleGenerator). It will exist until will merged to original repository!!!

## 🚀 How to Use

1. **Open the menu**  
2. **Select your rank**  
   - The maximum rank is **Ace**.  
   - If you need something more advanced like **Master** or **Champion**, it’s better to create those manually.  
3. **Drop your actor**  
   - Preferably a **Pokémon**.  
   - It also works with **Trainer NPCs**, but requires some adjustments with the special attribute.  
4. **Generate your Pokémon/Trainer** 🎉  

---

## ✨ Features

This generator can automatically create and assign:

- **Attributes** 
- **Social Attributes** 
- **Skills**  
- **Gender** – (Only male and female. If you need something specific or a pokemon is only female or genderless do it manually after generation.)
- **Natures** – 


## 📌 Notes

- This module is designed to help with quick improvisation during roleplay sessions.  
- It works with both Pokémon and Trainer actors, though Trainers may need extra customization.  
- The goal is to have instant resources ready when players do something unexpected.
